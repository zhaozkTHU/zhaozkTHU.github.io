vector

vector是封装了动态大小数组的容器，能够存放各种类型的对象

基本函数

构造函数

  • vector():默认构造函数，创建空vector，长度为0
  • vector(int size):创建元素个数为size的vector
  • vector(int size,const t& t):创建vector，元素个数为size，每个均为t
  • vector(const vector&):拷贝构造函数

增加函数

  • push_back(const T& x)：向量尾部增加元素x
  • insert(iterator it, const T& x):向量中迭代器指向的元素前增加一个元素x
  • insert(iterator it, int n, const T& x):增加n个相同的元素x
  • insert(iterator it,const_iterator first,const_iterator last):迭代器指向元素前插入另一个相同类型向量[first,last)间的数据

删除函数

  • erase(iterator it):删除向量中迭代器指向元素
  • erase(iterator first, iterator last):删除区间内的元素
  • pop_back():删除最后一个元素
  • clear():清除所有元素

遍历函数

  • at(int pos):返回pos元素位置的引用
  • front():首元素的引用
  • back():末尾元素的引用
  • begin():首元素的指针
  • end():末尾元素的指针

其他函数

  • bool empty() const:判断向量是否为空
  • size():向量中元素的个数
  • capacity():向量所能容纳最大元素值
  • swap(vector&):交换两个同类型向量的数据
  • void assign(int n, const T& x):设置前n个元素为x
